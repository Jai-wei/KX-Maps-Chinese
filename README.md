# KX-Maps-Chinese
激战2 KX 地图 汉化

科技下载链接:[KX Trainer](https://krixx.xyz/),这些地图文件能够更好的帮助你探索游戏内的世界！

## 使用
使用这些文件必须要购买KX(即付费版本),并将本存储库的Maps文件替换到KX科技下的Maps文件夹内
